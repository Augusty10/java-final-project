# 🚀 Java Projects Collection  

This repository contains two Java-based projects developed as part of my learning journey:  

---

## 1️⃣ Java Chat Application  
Objective:
A real-time peer-to-peer chat system using Java, Socket Programming, and Threads.  
Features:  
- Server-client architecture for message handling  
- Multi-client support with threads  
- Simple JavaFX-based GUI  
- Group & private messaging support  
- Nicknames and connection logs  
- Basic message encryption  

How to Run: 
1. Start the Server (`ServerMain.java`).  
2. Run multiple Clients (`ClientMain.java`) and connect using the server IP.  
3. Start chatting in real-time!  


---

## 2️⃣ Blockchain Simulation (Java)  
Objective:
A simplified blockchain simulator to demonstrate core concepts of cryptocurrency mining and transactions.  

Features: 
- Block creation and hashing  
- Proof-of-Work (mining simulation)  
- Transaction creation and validation  
- Basic wallet & balance system  
- Console-based interface  

How  to Run: 
1. Run `BlockchainDemo.java`.  
2. Observe transaction creation, block mining, and balance updates in console output.
